# Agent configuration for sambaths-site

## Agent skills

### Issue tracker

Hybrid: code bugs/features on GitHub (`sambaths/sambaths-site`), content/PRDs/planning as local markdown under `.scratch/`. See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical roles with two overrides: `ready` (for AFK-ready) and `manual-review` (needs human). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context. One `CONTEXT.md` at root + `docs/adr/` for decisions. See `docs/agents/domain.md`.
